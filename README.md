
# ThouTube - A YouTube Clone

ThouTube is a video streaming website inspired by YouTube. Currently this project is only a front-end clone.

## Features

- Light/dark mode toggle
- Watch videos
- View Channel
- Search videos
- Different video categories
- Real time trending videos
- Mobile responsive

## Built With

- ReactJS
- React router
- CSS
- Rapid API 


## Environment Variables

To run this project, you will need to add the following environment variables to your `.env` file

`REACT_APP_RAPID_API_KEY`
